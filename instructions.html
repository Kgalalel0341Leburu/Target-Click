Target Click Game

Game Instructions

Game Goal

The goal of the game is to click targets as they appear on the screen before they disappear.

Each successful click increases your score. Missing targets or clicking in the wrong place may reduce your score.

How to Play

  1. Enter your player name on the launcher page.
  2. Select a difficulty level.
  3. Select the game length.
  4. Choose a target style (circles, squares, or mixed).
  5. Select optional features such as sound or penalties.
  6. Click Open Game Window.
  7. In the game window, click Start Game.
  8. Click targets as quickly as possible when they appear.
  9. Avoid clicking empty areas if penalties are enabled.

Difficulty Levels

  • Easy: targets appear slowly and stay visible longer
  • Medium: moderate speed and visibility time
  • Hard: targets appear quickly and disappear faster

Buttons in the Game Window

  • Start Game starts a new game
  • Pause / Resume pauses or resumes the game
  • Save Session stores the current game state
  • Load Session restores a saved session
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Each target hit adds points
  • If Double points is enabled, each hit scores double
  • If penalties are enabled, clicking empty space reduces your score
  • Faster reactions result in higher scores
  • The game ends when the timer reaches zero

Tips

  • Keep your mouse near the centre of the screen
  • Watch for targets appearing in different positions
  • On hard mode, react quickly as targets disappear fast
